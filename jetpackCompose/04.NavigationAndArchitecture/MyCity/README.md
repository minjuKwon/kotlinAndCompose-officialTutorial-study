# 프로젝트 내용
- 다람쥐 도시라는 가상 도시의 명소와 음식점을 추천해주고 즐겨찾기 할 수 있는 어플.
- 해상 장소 클릭 시 상세 화면 이동.
- [Ideogram](https://ideogram.ai)으로 그림 생성.

- 기기 화면 크기에 따른 동적 화면 제공
